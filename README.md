## 2019-1 캡스톤 디자인 프로젝트 클라이언트 앱(안드로이드)

### 팀원
김수빈, 손낙원, 이종희, 이찬주, 정채연, 최규현

### 앱 소개
본 앱의 이름은 '커피 공화국'으로써, <br>
사용자는 자신의 위치에서 원하는 카페를 검색하여 메뉴를 미리 주문하고, <br>
음료가 준비되면 카페를 방문하여 테이크아웃할 수 있다.
